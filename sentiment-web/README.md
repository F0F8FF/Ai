# Korean Sentiment Analyzer

A web-based Korean text sentiment analysis application powered by BERT.

## Features

- Real-time sentiment analysis
- Korean language support
- Web interface
- Probability score for predictions
- Based on BERT multilingual model
- 

## Tech Stack

- Flask (Web Framework)
- PyTorch (Deep Learning)
- Transformers (Hugging Face)
- BERT (multilingual-cased-nsmc)
- HTML/CSS/JavaScript (Frontend)

## Requirements

- Python 3.8+
- PyTorch 2.0+
- Transformers 4.30+
- Flask 2.0+
