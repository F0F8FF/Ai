# Voice Assistant

A voice-controlled assistant that recognizes speech commands and performs various tasks.

## Features
- Speech recognition and text conversion
- Time and date checking
- Web searching
- Voice memo saving
- Various greeting responses

## Installation

### Prerequisites
- Python 3.x
- pip (Python package installer)

## Available Commands
- "안녕" (Hello) - Responds with a greeting
- "시간" (Time) - Shows current time
- "날짜" (Date) - Shows today's date
- "검색 [keyword]" (Search [keyword]) - Performs Google search
- "메모 [content]" (Memo [content]) - Saves voice memo
- "도움말" (Help) - Shows command list
- "종료" (Exit) - Exits program

## Usage
1. Run the program:
2. Speak commands in Korean
3. The assistant will respond accordingly

## Features in Detail
- **Speech Recognition**: Uses Google's speech recognition API
- **Dynamic Responses**: Random selection from multiple greeting messages
- **File Operations**: Saves memos to 'voice_memos.txt'
- **Web Integration**: Opens web browser for searches
- **Time Management**: Provides current time and date information
