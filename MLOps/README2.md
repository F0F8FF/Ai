# ML Model Monitoring System

This project implements a machine learning model monitoring system using FastAPI, Prometheus, and Grafana.

## Features

- ML model serving with FastAPI
- Real-time metrics collection with Prometheus
- Visualization dashboard with Grafana
- Docker containerization

## Tech Stack

- FastAPI
- Prometheus
- Grafana
- Docker & Docker Compose
- Python 3.9
- Scikit-learn
