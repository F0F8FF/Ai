# WordRNN Text Generator

A simple text generation project using a word-based RNN with LSTM in PyTorch.

## Features
- Generates text based on input text data.
- Uses an LSTM network for sequential word prediction.
- Supports customizable vocabulary and embedding sizes.

## Requirements
- Python 3.8+
- PyTorch 1.9+
- torchtext 0.10+ (optional, for additional text processing)
- numpy
